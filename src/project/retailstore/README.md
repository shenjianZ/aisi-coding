### 1. 介绍
- 这是一个基于Qt框架的C++项目，实现了基于Qt的桌面端，使用Qt+MySQL实现的一个小项目。


**技术栈**:

> 使用到的技术 (technology）
> - Qt： 6.7.1 （Qt Core、Qt QUI、Qt Widgets 、Qt Sql）
> - MySQL 8.0.27
> 

**项目环境**：

- `Clion 2024.1.2`
- `Qt 6.7.1`
- `CMake 3.28.1`
