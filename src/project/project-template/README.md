### 1. 介绍
- 这是一个基于Springboot+Vue的模板项目，包含前后端代码，可以快速搭建一个前后端分离的项目。


**技术栈**:

- **后端**：`SpringBoot 3`+ `MySQL` + `mybatis-plus` + `Redis` + `RabbitMQ` + `Jwt`（基于Token的权限校验）

- **前端**：`Vue3` + `Vite `+ `ElementPlus` (深色模式适配)

**项目环境**：

- `JDK 17`
- `Maven`
- `Springboot 3.1.2`
- `InteliJ IDE 2023.4`




### 2. 项目结构
    ├── aisi-frontend             // 前端模块
    ├── base-config               //后端模块
