## 基于Vuepress theme hope搭建的博客+文档

### 线上地址：[爱思文档](https://blog.shenjianl.cn/)

 1. 博客
2. 文档
3. 简历

### 拉取代码
```bash
  git clone https://github.com/shenjianl/aisi-blog.git
```

### 安装依赖
```bash
  cd aisi-blog
  npm install
```
### 启动服务
```bash
npm run docs:dev
```

### 构建项目
```bash
npm run docs:build
```
